# 🎰 Costa Roulette - Analisador Premium de Roleta

O **Costa Roulette** é um analisador avançado de roleta com foco em dispositivos móveis, apresentando um design minimalista, escuro e de alta performance. O aplicativo emprega modelos estatísticos e matemáticos refinados (como Cadeias de Markov, Espaçamento do Crupiê, Terminais Vizinhos Físicos e Setores do Cilindro) para projetar tendências balísticas na roleta com precisão cirúrgica.

---

## 🚀 Funcionalidades de Destaque

*   **🌱 Fase de Germinação Avançada (50 Giros):** 
    O sistema aguarda e processa o histórico retrospectivo de no mínimo **50 números** antes de liberar sinais automáticos e previsões balísticas complexas. Isso assegura máxima fidelidade matemática na calibração inicial dos 14 motores integrados.
*   **📊 Estatísticas e Placar de Assertividade Inteligente:**
    *   **Acertos e Erros (Greens & Reds):** Só são contabilizados nos momentos em que o indicador de análise acendeu de fato (sinal ativo ou jogada confirmada).
    *   **Taxa de Assertividade (%):** Continua calculando de forma inteligente com base em todos os giros previstos no histórico global, oferecendo uma visão analítica contínua do comportamento da mesa.
*   **⚡ Resposta Tátil e Latência Zero:**
    Otimização no processamento do teclado numérico e reaproveitamento do contexto de áudio em nível de sistema, garantindo feedback sonoro instantâneo e livre de lentidões durante a digitação rápida.
*   **🎯 Filtro de Motores de IA:**
    Alternador dinâmico entre o modo **Essencial** (motores chave) e o modo **Completo** (todos os 14 motores analíticos em execução paralela).
*   **📱 Interface Ultra-Glow Mobile:**
    Aparência cyberpunk escura premium com detalhes em dourado e cinza-grafite, transições fluidas de abas animadas com `motion` e visualizadores gráficos de zonas (Vizinhos do Zero, Orfãos, Tier, etc.).

## 🛠️ Tecnologias Utilizadas

*   **Framework Principal:** [React 19](https://react.dev/)
*   **Build Tool e Dev Server:** [Vite](https://vite.dev/)
*   **Estilização:** [Tailwind CSS v4](https://tailwindcss.com/)
*   **Animações:** [Motion para React](https://github.com/motiondivision/motion)
*   **Ícones:** [Lucide React](https://lucide.dev/)
*   **Processamento de Imagem:** [Tesseract.js](https://tesseract.projectnaptha.com/) (Suporte a OCR de telas de roleta)

## 📦 Como Instalar e Rodar o Projeto Localmente

Siga os passos abaixo para executar o projeto em sua máquina de desenvolvimento:

1.  **Clonar o Repositório:**
    ```bash
    git clone https://github.com/seu-usuario/nome-do-repositorio.git
    cd nome-do-repositorio
    ```

2.  **Instalar as Dependências:**
    ```bash
    npm install
    ```

3.  **Iniciar o Servidor de Desenvolvimento:**
    ```bash
    npm run dev
    ```

4.  **Acessar a Aplicação:**
    Abra seu navegador e digite o endereço configurado pelo Vite (geralmente [http://localhost:3000](http://localhost:3000) ou o endereço indicado no terminal).

## 📋 Scripts Disponíveis

No arquivo `package.json` você encontrará os seguintes comandos úteis:

*   `npm run dev`: Executa a aplicação localmente em modo de desenvolvimento.
*   `npm run build`: Compila e gera os arquivos estáticos de produção na pasta `dist/`.
*   `npm run lint`: Valida o código utilizando as regras do compilador TypeScript (`tsc --noEmit`).
*   `npm run clean`: Remove os diretórios temporários de compilação.
